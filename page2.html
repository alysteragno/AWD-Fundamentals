1. Variables

Definition: Variables are used to store data that can be used and manipulated later in a program.

let age = 25;

This is an example of declaring a variable in JavaScript where the variable age is storing the number 25.

2. Control Structures

Definition: Control structures allow the program to make decisions and repeat actions.

if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

This code checks if the variable age is greater than or equal to 18. If true, it prints "You are an adult." Otherwise, it prints "You are a minor."

3. Functions

Definition: Functions are blocks of code that can be reused to perform specific tasks.

function greet(name) {
    return "Hello, " + name + "!";
}

This is an example of a simple function called greet. It takes one parameter, name, and returns a greeting message.

4. Data Structures

Definition: Data structures allow us to store and organize data efficiently for easier access and manipulation.

let person = {
    name: "John",
    age: 30,
    profession: "Developer"
};

This example shows a simple object person, which is a data structure containing multiple properties (name, age, profession).
